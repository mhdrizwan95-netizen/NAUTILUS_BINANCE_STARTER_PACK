# Feature Flags

Quick index of the most useful environment flags. Start modules in dry‑run where available, validate metrics, then flip live with conservative sizing.

## Core
- `TRADING_ENABLED` — master switch for live order placement
- `MIN_NOTIONAL_USDT`, `MAX_NOTIONAL_USDT` — notional rails for orders
- `MAX_ORDERS_PER_MIN` — rate limiter
- `WARMUP_SEC` — block entries for N seconds after start/reconnect

## Strategy (MA + HMM)
- `STRATEGY_ENABLED`, `STRATEGY_DRY_RUN`
- `STRATEGY_SYMBOLS`, `STRATEGY_INTERVAL_SEC`, `STRATEGY_QUOTE_USDT`
- `ENSEMBLE_ENABLED`, `ENSEMBLE_WEIGHTS`, `ENSEMBLE_MIN_CONF`
- `HMM_ENABLED`, `HMM_MODEL_PATH`, `HMM_WINDOW`, `HMM_SLIPPAGE_BPS`

## Event Breakout
- `EVENT_BREAKOUT_ENABLED`, `EVENT_BREAKOUT_DRY_RUN`, `EVENT_BREAKOUT_METRICS`
- `EVENT_BREAKOUT_SIZE_USD`, `EVENT_BREAKOUT_HALF_SIZE_MINUTES`
- `EVENT_BREAKOUT_ATR_TF`, `EVENT_BREAKOUT_ATR_N`, `EVENT_BREAKOUT_TRAIL_ATR_MULT`
- `EVENT_BREAKOUT_DENYLIST_ENABLED`, `EVENT_BREAKOUT_DENYLIST_PATH`
- `ENTROPY_DENY_ENABLED`, `ENTROPY_DENY_REASON_COUNT`, `ENTROPY_DENY_WINDOW_MIN`, `ENTROPY_DENY_DURATION_MIN`

## Guards
- Depeg: `DEPEG_GUARD_ENABLED`, `DEPEG_THRESHOLD_PCT`, `DEPEG_CONFIRM_WINDOWS`, `DEPEG_ACTION_COOLDOWN_MIN`, `DEPEG_EXIT_RISK`, `DEPEG_SWITCH_QUOTE`
- Funding: `FUNDING_GUARD_ENABLED`, `FUNDING_SPIKE_THRESHOLD`, `FUNDING_TRIM_PCT`, `FUNDING_HEDGE_RATIO`

## Protection & Execution
- Stop Validator: `STOP_VALIDATOR_ENABLED`, `STOP_VALIDATOR_INTERVAL_SEC`, `STOP_VALIDATOR_GRACE_SEC`, `STOP_VALIDATOR_REPAIR`, `STOPVALIDATOR_REPAIR_TP`
- Stop repair ping: `STOPVAL_NOTIFY_ENABLED`, `STOPVAL_NOTIFY_DEBOUNCE_SEC`
- Auto cutback/mute: `AUTO_CUTBACK_ENABLED`, `AUTO_CUTBACK_SIZE_MULT`, `AUTO_CUTBACK_DURATION_MIN`, `AUTO_MUTE_SCALP_THRESHOLD`, `AUTO_MUTE_SCALP_WINDOW_MIN`, `AUTO_MUTE_SCALP_DURATION_MIN`
- Risk parity: `RISK_PARITY_ENABLED`, `RISK_PARITY_TF`, `RISK_PARITY_N`, `RISK_PARITY_MIN_NOTIONAL_USD`, `RISK_PARITY_MAX_NOTIONAL_USD`
- Min‑notional block: `MIN_NOTIONAL_BLOCK_USD`

## Telemetry & Notifiers
- Telegram: `TELEGRAM_ENABLED`, `TELEGRAM_BOT_TOKEN`, `TELEGRAM_CHAT_ID`
- Telegram bridge: `TELEGRAM_BRIDGE_ENABLED` (relay BUS `notify.telegram`)
- Health TG: `HEALTH_TG_ENABLED`, `HEALTH_DEBOUNCE_SEC`
- Digest: `DIGEST_INTERVAL_MIN`, `DIGEST_INCLUDE_SYMBOLS`, `DIGEST_6H_ENABLED`, `DIGEST_6H_BUCKET_MIN`, `DIGEST_6H_MAX_BUCKETS`

## WS & Fills
- WS fills listener: `EXEC_FILLS_LISTENER_ENABLED`
- WS health: `WS_HEALTH_ENABLED`, `WS_DISCONNECT_ALERT_SEC`, `WS_RECONNECT_BACKOFF_MS`

## Execution Defaults
- Taker slippage caps: `SPOT_TAKER_MAX_SLIP_BPS`, `FUT_TAKER_MAX_SLIP_BPS`
- Maker shadow: `SCALP_MAKER_SHADOW`, `MAKER_PRICE_IMPROVE_BPS`
